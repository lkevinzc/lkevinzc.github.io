Model Serving made Efficient in the Cloud (MOSEC)

Mosec is a high-performance ML model serving framework built with a fast Rust web layer. It supports all different ML frameworks, such as Jax, PyTorch, TensorFlow, etc., and is super easy to write the server codes. The main features include dynamic batching and CPU/GPU pipelines to fully exploit your compute machine.
